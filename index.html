swfit问题总结一

swift项目桥接OC第三方库 言归正传，先上图 –> 创建.h文件 在本项目的taget下路径：target->Build Setting->Search Path->User Header Search Paths 设置目录的路径${SRCROOT},然后选择recursive 3. 在本项目的taget下路径：target->Build Setting->Swift Compiler - General ->Objective-C Bridging Header 双击输入$(SRCROOT)/本项目名称/xxx.h 在Pod的taget下路径：target->Build Setting->Search Path->User Header Search Paths 设置目录的路径${SRCROOT},然后选择recursive 这样就大功靠成了，可以在桥接文件随便import第三方库的东西啦！！！
